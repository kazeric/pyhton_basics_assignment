# Python Basics Assignment 

## Overview

This assignment covers fundamental Python programming concepts through interactive notebooks and hands-on coding challenges. The materials progress from basic variables and data types through control flow, functions, and object-oriented programming.

## Contents

### Instructional Notebooks

These Jupyter notebooks provide explanations and examples for each topic:

| Notebook                | Topic                  | Key Concepts                                                    |
| ----------------------- | ---------------------- | --------------------------------------------------------------- |
| `1_Variables.ipynb`     | Variables & Assignment | Variable declaration, naming conventions, variable reassignment |
| `2_data_types.ipynb`    | Data Types             | int, float, str, bool, type conversion, type checking           |
| `3_datatypes.ipynb`     | Collections            | Lists, tuples, sets, dictionaries, indexing, slicing, iteration |
| `4_loops.ipynb`         | Loop Structures        | for loops, while loops, break, continue, nested loops           |
| `5_control_flows.ipynb` | Conditionals           | if/elif/else, logical operators (and, or, not), comparison      |
| `6_function.ipynb`      | Functions              | Function definition, parameters, return values, scope           |

## Learning Objectives

After completing this assignment, you will be able to:

Declare and use variables effectively  
Work with all basic Python data types  
Manipulate collections (lists, dictionaries, sets, tuples)  
Control program flow with loops and conditionals  
Write reusable functions with proper parameters and returns  
Design and implement simple classes with methods  
Understand object-oriented programming principles

## How to Use This Assignment

### Step 1: Study the Notebooks

Read through each notebook (1-6) sequentially to understand the concepts.

### Step 2: Complete the Challenges

After each notebook, complete the corresponding challenge file to reinforce your learning.

### Step 3: Explore Classes

Review `7_classes.py` to see practical applications of OOP concepts.

### Step 4: Test Your Code

Run your challenge files to verify they work correctly:

```bash
python 1_challenge.py
python 2_challenge.py
# ... and so on
```
